# experiences

This folder is a small travel-writing system: `travel.md` is the trip index, `events.md` is the event index, and `places/` holds the actual narrative pages. `places/README.md` is the geography browser.

- `# place`
- `## trip subtitle` for shared multi-stop trips
- `### YYYY-MM-DD` for day blocks
- `#### HHMM` for finer sequence when needed
- inline images with short italic captions
- `#draft` at the bottom if incomplete

- write in first person, mostly present tense
- keep it lived-in and concrete; do not turn it into summary prose
- movement between places stays in the prose with relative date-anchored links
- same-day detours use bidirectional backlinks on the same date and resume with `---`
- later revisits to the same place also use `---`
- `travel.md` and `events.md` stay light; the real substance lives in place pages
- in `places/README.md`, keep the existing country or region grouping
- display place names in the language or script of the culture of the place
- order places alphabetically by their English slug or link target, not by the display script
- prefer the newer normalized pages like York, Rajasthan, Laos/Cambodia, and Spiti/Kinnaur over older rougher files
