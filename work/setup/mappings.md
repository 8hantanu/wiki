# mappings

I use [Karabiner](https://karabiner-elements.pqrs.org/) on macOS and [PowerToys](https://learn.microsoft.com/en-us/windows/powertoys/) on Windows to remap two keys that always felt wasteful in their default positions:

- <kbd>Caps Lock</kbd> → <kbd>Left Ctrl</kbd>
- <kbd>Right Shift</kbd> → <kbd>Backspace</kbd>

The idea is to put the most-used modifier and correction keys where my pinkies already rest, instead of stretching to the corners of the board.
